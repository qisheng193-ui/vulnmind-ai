# Xiaomi Agent 商店文案

## 名称

VulnMind AI Secure Code Review Agent

## 一句话介绍

面向代码仓库的 AI 安全审计助手，帮助研发团队快速发现高风险调用点并给出修复建议。

## 详细描述

VulnMind AI 是一个用于代码安全左移的审计 Agent。它能够遍历 PHP、Python、JavaScript 源码仓库，识别动态执行、命令执行、不安全反序列化、动态文件包含等高风险模式，并自动生成结构化风险报告、验证建议和修复建议。

## 核心能力

- 代码仓遍历与风险点定位
- 高风险模式识别与规则化分级
- 审计报告输出
- 修复建议生成

## 适用对象

- 企业研发团队
- 安全工程师
- 代码审计团队
- DevSecOps 场景

## 建议标签

- 安全审计
- 代码扫描
- DevSecOps
- 风险巡检
