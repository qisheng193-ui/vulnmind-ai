# 小米 Agent 申请与发布步骤

更新时间参考：2026-04-21 公测公告、2026-04-28 发布操作指南

## 1. 准备开发者账号

1. 注册或登录小米开发者账号。
2. 完成个人或企业开发者认证。
3. 若要申请 Agent 公测，优先准备企业开发者资料更稳妥。

## 2. 申请 Agent 权限

1. 发送邮件到 `developer@xiaomi.com`。
2. 邮件标题使用：`Agent 生态申请 - 开发者名称`
3. 邮件正文至少包含：
   - 开发者名称
   - 小米 ID
   - 团队主账号小米 ID（如果有）
   - 申请服务类型：`Agent`
   - 服务场景/功能说明

## 3. 获得权限后创建 Agent

1. 登录小米澎湃OS开发者平台管理中心。
2. 点击“创建Agent应用”。
3. 填写名称、描述、图标。

限制建议：

- 名称不超过 30 个字符
- 描述不超过 100 个字符
- 图标 256x256 px

## 4. 导入或编辑 workspace

你有两种方式：

1. 在平台里直接自然语言生成 Agent，然后手动修改 `Prompt / Config / Profile / Skill`
2. 先创建空白 Agent，再把本项目里的 `workspace_template` 内容按官方 ZIP 结构替换后上传

## 5. 预览和调试

1. 点击“预览”进入云真机。
2. 在手机端或 Miclaw 中安装测试包。
3. 验证 Agent 名称、介绍、报告格式和交互效果。

## 6. 提交审核

1. 点击“发布”。
2. 确认名称、描述、图标、版本内容。
3. 提交审核。
4. 若 `config.json` 报错，优先检查 `id/name/description/type/version` 是否为字符串且字段名拼写正确。

## 7. 审核周期

官方文档写明一般为 `1~3 个工作日`。
